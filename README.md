# Bakery Shop Landing Page

Учебный проект лендинга магазина выпечки. Это мой самый первый проект, на котором я отрабатывал базовые навыки верстки.

## Цель проекта
Закрепить теоретические знания HTML и CSS на практике, создав визуально приятную и структурно правильную страницу.

## Технологии
*   **HTML5**: использование семантических тегов (`<header>`, `<main>`, `<footer>`, `<section>`).
*   **CSS3**: работа с позиционированием, кастомными шрифтами и стилизацией форм.

## Чему я научился
*   Работать с блочной моделью.
*   Подключать и настраивать внешние шрифты.
*   Верстать интерфейс по макету.

## Текущее состояние
*   **Адаптивность**: Страница оптимизирована только под **десктопные экраны**.
*   **Код**: Это мой первый опыт, поэтому архитектура стилей может быть не идеальной, но проект полностью отражает мой прогресс на старте обучения.
