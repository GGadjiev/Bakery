Мой первый учебный проект — лендинг магазина выпечки, созданный на чистом HTML и CSS. 
Проект помог мне на практике освоить базовые принципы вёрстки: работу с семантическими тегами, подключение шрифтов, позиционирование элементов и стилизацию. 
Адаптивная вёрстка не реализована — страница оптимизирована под десктоп. 
Код не претендует на идеальность, но отражает мои первые шаги в веб-разработке.
